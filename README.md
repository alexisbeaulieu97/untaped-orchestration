# untaped-orchestration

`untaped-orchestration` is a planned standalone CLI for Git-native, typed
orchestration stores and bounded agent bootstrap briefs. Canonical state stays
human-readable under `.untaped/orchestration/`; agents query and mutate it
through deterministic CLI contracts.

Status: **specification only**. No implementation has landed.

See the proposed
[v1 design specification](docs/superpowers/specs/2026-07-09-orchestration-v1-design.md).
